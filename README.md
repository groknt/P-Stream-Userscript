# Userscript
Userscript replacement for the P-Stream extension

1. Install a user script extension (e.g., Violentmonkey)
2. [Download](https://raw.githubusercontent.com/groknt/P-Stream-Userscript/main/p-stream.user.js) the script!
3. Enable it for P-Stream!
